## FSB(Format String Bug)

### example1

```c
// Name : example1
// Compaile : gcc -o example1 example1.c -m32 -mpreferred-stack-boundary=2

#include<stdio.h>

void main()
{
	int a = 20;
	int b = 21;
	int c = 22;
	int d = 23;
	printf("%p %p %p %p %p %p %p %p\n");
}
```

__Printf__ 함수는 Printf("%d", val); 와 같이 `포맷 스트링("%d")`과  해당 포맷에 대응되는 `인자`의 형태로 사용됩니다.

하지만 __example1__ 처럼 대응 되는 인자없이 `printf("%p %p %p %p %p %p %p %p\n");` 같이 사용되었다면
