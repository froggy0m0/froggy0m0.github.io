## pwd

`pwd` 명령어는 현재 어떤 디렉토리에 있는지를 확인할 수 있다.

```
[trainer2@ftz trainer2]$ pwd
/home/trainer2
```

## cd

### cd .. 

`cd ..`명령어는 현재 디렉터리에서 한단계 위로 이동해준다. 즉 현재 /home/trainer2 에서 /home으로 이동해준다.여기서 다시 한단계 위로 이동하면경로는 /가 되고 이것을 '최상위로 이동했다.'라고 한다.

리눅스에서는 /를 최상위 디렉토리 또는 루트 디렉토리라고 한다.

```
[trainer2@ftz trainer2]$ cd ..
[trainer2@ftz home]$ pwd
/home
```

### cd

`cd` 명령어는 내려갈 디렉토리명를 입력하여 해당 디렉토리로 내려간다.

혹은 cd/ home/trainer2 와같이 절대경로를 입력해도된다.

```
[trainer2@ftz home]$ cd trainer2
[trainer2@ftz trainer2]$ pwd
/home/trainer2
```

### cd /

`cd /` 명령어는 최상위 디렉토리로 이동한다.

```
[trainer2@ftz trainer2]$ cd /
[trainer2@ftz /]$ pwd
/
```



## mkdir    

`mkdir `현재경로에서 새로운 디렉토리를 생성

``` 
[trainer2@ftz trainer2]$ mkdir linuxer
```

## rmdir

`rmdir` 디렉토리 삭제

```
[trainer2@ftz trainer2]$ rmdir linuxer
```

## cp 

`cp` 파을을 복사한다. cp _파일1 파일2_ 은 파일1이 파일2 라는 이름으로 복사된다는뜻이다 

```
[trainer2@ftz trainer2]$ cp 복사원본 복사사본
```

## rm

`rm` 파일을 삭제

```
[trainer2@ftz trainer2]$ rm 복사사본
```

## mv

`mv` cp명령과는 달리, 파일이 복사된 후 원본 파일은 자동으로 삭제가 됩다..

따라서 이 mv 명령은 파일 이름을 변경할 때 많이 쓰이게된다. .

```
[trainer2@ftz trainer2]$ mv 복사원본 복사사본
```
