## passwd

```
[trainer6@ftz trainer6]$ cat /etc/passwd
root:x:0:0:root:/root:/bin/bash
bin:x:1:1:bin:/bin:/sbin/nologin
daemon:x:2:2:daemon:/sbin:/sbin/nologin
...
```

### cat

리눅스 기본 명령어로 'cat 파일이름' 을 입력하면 파일의 내용을 화면으로 출력해주는 역활을 한다.

### 패스워드파일 

한 서버를 사용하는 사용자들의 모든 정보를 기록해 놓은 파일

<span style = "font-size: 35px">__<span style="color:maroon">root</span>:<span style="color:olive">x</span>:<span style="color:green">0</span>:<span style="color:Orange">0</span>:<span style="color:royalblue">Admin</span>:<span style="color:#5800fd">/root</span>:<span style="color:RED">x</span>__</span>

`:` 문자는 필드를 구분해주는 문자이다. 위와같이 _passwd_ 파일에는 7개의 필드로 나누어져 있다.

1. __<span style="color:maroon">root</span>__ : 서버에 로그인할 때 사용되는 아이디(계정)

1. __<span style="color:olive">x</span>__ : 패스워드(암호화된 문자열로 나타냈지만 현재는  /etc/shadow 파일에 저장, 관리자권한만 볼 수 있다.)

1. __<span style="color:green">0</span>__ : 아이디(컴퓨터가 읽기 쉽게 숫자를 사용하여 사용자를 판단)

1. __<span style="color:Orange">0</span>__ : 그룹

1. __<span style="color:royalblue">Admin</span>__ : 사용자의 이름

1. __<span style="color:#5800fd">/root</span>__ : 해당 사용자가 로그인에 성공 했을 떄 기본으로 위치하게 되는 디렉토리

1. __<span style="color:RED">x</span>__ : 처음 로그인 했을 떄 실행되게 할 프로그램 ex) 쉘 실행
