## 파일의 종류

### 일반 텍스트 파일(txt)

`.txt` 글자로만 이루어진 파일 _cat_ 명령을 통해 출력할 수 있다.

#### 텍스트 파일 생성 법

```
# 텍스트 파일 작성
[trainer8@ftz trainer8]$ cat > test.txt
hello~ test
# Ctrl + D 입력으로 종료

...

# 텍스트 출력
[trainer8@ftz trainer8]$ cat test.txt
hello~ test
[trainer8@ftz trainer8]$
```

1. cat > 파일이름.txt
2. 원하는 내용 작성
3. Ctrl + D로 종료.

`>` 문자는 __리다이렉션__ 이라고 읽으며 __방향을 전환한다__ 라는 뜻을 가지고 있다.

#### 기존 파일에 이어서 쓰기

```
# 텍스트 이어 쓰기
[trainer8@ftz trainer8]$ cat >> test.txt
test hello~~~!
# Ctrl + D 입력으로 종료

...

# 텍스트 출력
[trainer8@ftz trainer8]$ cat test.txt
hello~ test
test hello~~~!
```

`cat > 기존파일이름.txt` 을 사용하여 작성하게 되면 기존 파일의 내용이 사라지게 된다

이때 _리다이렉션_  두번 사용하여 텍스트 파일에 내용을 덧붙일 수 있다.

_cat_ 명령어는 모니터로 그대로 출력 해줘서 ___>_ (리다이렉션)__ 을 통해서 출력 방향을 전환시켜 이후에 입력된 파일이름으로 보내라는 의미로 해석할 수 있다.

### 프로그램 소스 파일

컴퓨터 언어로 입력한 파일로 _컴파일_ 이라는 과정을 거쳐 실행

```
## 1. 소스코드 작성  (10 + 20 을 계산하는 출력 프로그램)
[trainer8@ftz trainer8]$ cat > program.c
int main()
{
int a=10,b=20;
printf("10 + 20 = %d",a+b);
}

# Ctrl + D 입력으로 종료

...

# 2.컴파일
[trainer8@ftz trainer8]$ gcc -o program.exe program.c

...

# 3.실행
[trainer8@ftz trainer8]$ /home/trainer8/program.exe
10 + 20 = 30
[trainer8@ftz trainer8]$
```

1. cat > 소스 파일이름.c 
2. gcc -o 프로그램 이름 소스 파일이름.c
3.  상대 경로 or 절대 경로/프로그램 이름

##### 절대경로사용법

`.`(현재 디렉터리를 의미) 를 통해 절대 경로를 설정할 수 있다.

_./program.exe_
