## 파일 압축

### tar

```
[trainer7@ftz 가사집]$ tar cvf songs.tar *
넥스트4집-가사
샵3집-가사
서태지2집-가사
...
[trainer7@ftz 가사집]$ tar xvf songs.tar
넥스트4집-가사
샵3집-가사
서태지2집-가사
[trainer7@ftz 가사집]$
```

명령어중 `*`는 해당디렉토리의 모든 파일을 의미한다.

* #### 명령어

`tar` 단순히 파일을 묶어버리는 용도 

__합치기__   :  tar cvf 합칠파일 합칠파일들

__해제하기__ :  tar xvf 해제할파일

* #### 옵션

| 옵션            | 설명                                       |
| --------------- | ------------------------------------------ |
| __c__ (Create)  | 새로운 파일을 만드는 옵션.                 |
| __x__ (eXtract) | 압축을 해제시키는 옵션.                    |
| __v__ (View)    | 압축이 되거나 풀리는 과정을 출력하는 옵션. |
| __f__ (File)    | 파일로서 백업을 하겠다는 옵션.             |

### gzip

```
[trainer7@ftz 가사집]$ gzip songs.tar
[trainer7@ftz 가사집]$
...
[trainer7@ftz 가사집]$ gzip -d songs.tar.gz
[trainer7@ftz 가사집]$
```

#### 명령어

`gzip` 한번에 한개의 파일만 압축할 수 있다.

_tar_ 과 _gzip_ 을 같이 사용할 수 도 있다.

__<span style="color:royalblue">_tar_ 을 사용하여 여러 파일을 1개로 묶고   </span> - ><span style="color:olive">         _gzip_ 을 사용하여 압축</span>__

__<span style="color:royalblue">                     (file.tar)                                 </span>  - >  <span style="color:olive">               (file.tar.gz)</span>__

* __압축하기__ : gzip 파일이름

* __압축해제__ : gzip -d 파일이름

#### 확장자명

| 확장자명   | 설명                                                         |
| ---------- | ------------------------------------------------------------ |
| __tar__    | _tar_ 프로그램을 사용하여 여러 파일들이 하나로 뭉쳐져 있는 파일 |
| __gz__     | _gzip_ 프로그램을 사용하여 압축된 파일                       |
| __tar.gz__ | _tar_ 프로그램을 사용하여 합쳐진 파일을 _gzip_ 프로그램을 사용하여 압축을 한 파일 |
| __tgz__    | _tar.gz_ 을 합쳐서 _tgz_ 라는 확장자로 만듬                  |
