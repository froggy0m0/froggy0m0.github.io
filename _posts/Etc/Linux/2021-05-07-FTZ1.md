## 프롬프트

리눅스를 처음 접속하면 `프롬프트`가 나타나며 이것은 서버에 성공적으로 로그인 되었다는 것을 의미한다.

[trainer1@<span style="color:red">ftz</span> <span style="color:blue">trainer1</span>]$

[접속ID@<span style="color:red">서버</span> <span style="color:blue">현재위치</span>]

`프롬프트`는 항상 대기상태에서 사용자의 명령을 기다린다.

## ls

* 현재 위치를 기준으로 컴퓨터에 설치된 폴더와 파일들을 보여주는 명령어

```
[trainer1@ftz trainer1]$ ls
addhtml      pulic_html     sh.c       t1.txt
addhtml.c             sh      start.txt  t2.txt
```

### -l 

`-l` 옵션을 사용하면 디렉토리와 파일을 구분하여 출력하게된다.

```
[trainer1@ftz trainer1]$ ls -l
-rwxr-x---  1  trainer1  trainer1  14698 Oct 23    2000   addhtml
-rw-rw----  1  trainer1  trainer1    713 Oct 23    2000   addhtml.c
drwxr-x---  2  trainer1  trainer1   4096 Feb 27    2002   public_html
```

첫줄은

* _addhtml_ 에 관한 정보이고 _Oct 23_ 은 생성된 날짜

* _14698_ 은 이 파일의 용량

* _trainer1 trainer1_ 이 부분은 Permission에대한 정보이다.

* _-rwxr-x---_ 은 파일의 mode와 파일의 성격에대한 정보이다.

  가장앞에 왼쪽의 문자가 _-_ 로 되어있다면 파일을 뜻한다.

* _drwxr-x---_ 처럼 가장왼쪽부분이 - 가 아닌 d 로 되어있다면

  그것은 디렉토리(directory)를 의미한다. 

### -a 

`-a` 옵션은 ls 명령으로 보이지않는 숨겨진 파일들(설정파일과 같은 중요한 파일)을 출력한다.   

```
[trainer1@ftz trainer1]$ ls -a
.             .bashrc         .gvimrc         .vimrc
..            .cshrc           .muttrc         addhtml
```

숨겨진 파일은 파일명 가장 앞부분에 `.` 이 들어가 있는 특징이있다.

보통 파일명 앞에 `.`을 붙이면 그 파일은 숨겨진 파일로 변하게 된다.

또한 `ls -al` 를 사용하면 숨겨진파일과 디렉터리 리스트가 출력된다.
