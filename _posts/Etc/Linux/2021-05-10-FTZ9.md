## 리눅스의 권한

리눅스는 각 사용자마다 특별한 권한을 부여하여 그 권한의 범위를 넘어서지 않는 파일들만 제어할 수 있도록 설정되어 있다.

만약 권한이 없는 파일을 보려고 하면 Permission Denied(집건 거부) 에러 메시지가 출력 된다.

### 권한(Permission)

_ls -al_ 명령을 통해서 파일들의 권한에 대한 정보를 출력 시킬 수 있다.

#### 사용자 정보

리눅스에서는 4 가지 종류의 사용자가 있다.

* __유저__

  자신을 의미

* __그룹__

  자신이 속한 그룹

* __아더__

  유저와 그룹을 제외한 모든 사용자

* __루트__

  절대적인 권한을 가지고 있는 사용자

```
[trainer9@ftz trainer9]$ id
uid=2009(trainer9) gid=2009(trainer9) groups=2009(trainer9)
[trainer9@ftz trainer9]$
```

#### id 명령어

* __uid__

  User ID의 약자이다. 

  2009는 컴퓨터가 알아보기 쉽게 지정된 숫자로 누군지 판단한다

* __gid__

  Group ID의 약자이다.

  다른 사람을 자신의 gid를 가진 그룹에 속 하게할 수 있다.

  특별한 일이 없는 한 gid는 항상 uid와 같다.

* __groups__

  현재 자신이 어떤 그룹에 속해 있는지를 의미

  기본으로 자신의 uid와 같은 그룹에 속하게 된다.

### 파일의 권한 정보

```
[trainer9@ftz trainer9]$ ls -al test1
-rwxrwxrwx    1 trainer9 trainer10        5 10월 20 21:35 test1
[trainer9@ftz trainer9]$
```

가장 왼쪽 부분의 <span style="color:red">-</span><span style="color:royalblue">rwx</span><span style="color:orange">rwx</span><span style="color:purple">rwx</span> 는

<span style="color:red">-</span> 부분은 디렉터리인지 파일 인지를 구분

그리고 순서대로 <span style="color:royalblue">유져의 권한</span> <span style="color:orange">그룹의 권한</span> <span style="color:purple">아더의권한</span> 으로 구성되어 있다.

r는 Read 읽기권한

w는 Write 쓰기권한

x는 eXecute 실행권한

<span style="color:red">-</span><span style="color:royalblue">rwx</span><span style="color:orange">rwx</span><span style="color:purple">rwx</span>    1 <span style="color:#5F9EA0">trainer9</span> <span style="color:#0064FF">trainer10</span>        5 10월 20 21:35 <span style="color:olive">test1</span>

1. <span style="color:olive">test1</span> 이라는 파일은...
2. <span style="color:#5F9EA0">trainer9</span> 라는 uid를 가진 사용자에게...
3. <span style="color:royalblue">rwx</span> (읽기 쓰기 실행)권한이 있고 ...
4. <span style="color:#0064FF">trainer10</span> 이라는 gid를 자긴 사용자게에게도 ...
5. <span style="color:orange">rwx</span> 권한이 있고...
6. <span style="color:#5F9EA0">trainer9</span> 와 <span style="color:#0064FF">trainer10</span> 를 제외한 아더 에게도
7. <span style="color:purple">rwx</span> 권한이 있다.

※ 리눅스에서는 확장자(.exe, .bat)만 보고 실행 파일인지 아닌 지를 판단 할 수 없다.

실행 파일인지 구분 하는 방법은 실행 권한인 'x' 라는 문자가 있는지로 판단하면 된다.
