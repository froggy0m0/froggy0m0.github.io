## Cron

`Cron`은 리눅스에서 사용되는 시간기반 작업 스케줄러이다.

Cron에 의해서 설정된 시간에 따라서 수행되게된다 

   

## CronTab

`Crontab`은 명령어를 사용하여 앞서말한 Cron을 관리할 수 있다.

```
# crontab -e
```

crontab -e 명령어는 현재 로그인된 사용자의 Cron을 편집할 수 있다.

root로 로그인한 경우 `/var/spool/cron/root`에 저장된다.

​    

```
01 01 * * * /home/user/test.sh
```

위와같이 편집하면 매일 1시 1분에 /home/user/test.sh 를 실행하게된다.

설정은 `분 시 일 월 요일 명령어` 형태로 되어있는데

분 : 0 ~ 59

시 : 0 ~ 23

일 : 1 ~ 31

월 : 1 ~ 12 OR jan ~ dec

요일 : 0 ~ 7 (0은 일요일) OR sun~sat

| 기호 | 설명                                             |
| ---- | ------------------------------------------------ |
| *    | 그 단위 전체를 설정                              |
| -    | 숫자 범위 설정 ex) 1-5 는 1~5분                  |
| ,    | 구분자로 설정  ex) 2,10-15 는2분,10~15분         |
| /    | 특정 단위로 건너 뛸때 설정  ex) */5 는 매5분마다 |



위의 표와같이 직접 설정하는것 외에도

​    

```
@hourly /home/user/test.sh
```

| 명령어    | 설명                               |
| --------- | ---------------------------------- |
| @reboot   | reboot후 1회                       |
| @yearly   | 매년 1월 1일 0시 0분 (= 0 0 1 1 *) |
| @annually | 매년 1월 1일 0시 0분 (= 0 0 1 1 *) |
| @monthly  | 매월 1일 0시 0분 (= 0 0 1 * *)     |
| @weekly   | 매주 일요일 0시 0분 (= 0 0 * * 0)  |
| @daily    | 매일 0시 0분 (= 0 0 * * *)         |
| @midnight | 매 자정마다 (= 0 0 * * *)          |
| @hourly   | 매시 0분 ( = 0 * * * *)            |

​    

```
# crontab -l
```

crontab -l 을 명령어를 사용하여 설정된 작업을 확인할 수 있다.
